# World Explorer - By Hana


**The most beautiful World & Ethiopia Quiz App made with love in Ethiopia!**

## Features
- Search all 195 countries & continents
- Special Africa menu → Eastern, Western, Northern, Central, Southern
- Beautiful centered country details with flag
- Global Leaderboard — see who is #1 in the world!
- Change your name & appear on leaderboard
- Works offline
- 100% made in Flutter by Hana
